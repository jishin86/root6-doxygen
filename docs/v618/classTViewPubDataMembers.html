TViewPubDataMembers Class Reference

View implementing the TList interface and giving access all the TDictionary describing public data members in a class and all its base classes without caching any of the TDictionary pointers.

Adding to this collection directly is prohibited. Iteration can only be done via the TIterator interfaces.
