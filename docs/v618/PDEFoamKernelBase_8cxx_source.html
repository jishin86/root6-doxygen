28 /*! \class TMVA::PDEFoamKernelBase
29 \ingroup TMVA
30 This class is the abstract kernel interface for PDEFoam. The
31 kernel can be used for manipulating (smearing) the cell values of a
32 PDEFoam, by passing it as an argument to
33 PDEFoam::GetCellValue(...).
34 
35 Derived classes must implement the Estimate() function to provide a
36 specific kernel behaviour.
37 */
