◆ GetAssocObj()

TObject * TDSetElement::GetAssocObj ( Long64_t  i,
Bool_t  isentry = kFALSE 
)

Get i-th associated object.

If 'isentry' fFirst is subtracted, so that i == fFirst returns the first object in the list. If there are not enough elements in the list, the element ilist_size is returned (if the list has only one element this only one element is always returned. This method is used when packet processing consist in processing the objects in the associated object list.
