TImage Class Reference
Graphics » 2D Graphics » Basic graphics

An abstract interface to image processing library.

It allows for the reading and writing of images in different formats, several image manipulations (scaling, tiling, merging, etc.) and displaying in pads.

The concrete implementation of this class is done by the TASImage class. The methods are documented in that class.
