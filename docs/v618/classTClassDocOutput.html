◆ WriteClassDescription()

void TClassDocOutput::WriteClassDescription ( std::ostream &  out,
const TString &  description 
)
protectedvirtual

Called by TDocParser::LocateMethods(), this hook writes out the class description found by TDocParser.

It's even called if none is found, i.e. if the first method has occurred before a class description is found, so missing class descriptions can be handled. For HTML, its creates the description block, the list of functions and data members, and the inheritance tree or, if Graphviz's dot is found, the class charts.
