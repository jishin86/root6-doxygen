void HtmlTranslateEscapes ( char *  z)

Translate escape sequences in the string "z".

"z" is overwritten with the translated sequence.

Unrecognized escape sequences are unaltered.

Example: 

 input  = "AT&amp;T &gt MCI"
 output = "AT&T > MCI"
