ROOT::Detail::RDF::RNodeBase Class Referenceabstract

Base class for non-leaf nodes of the computational graph.

It only exposes the bare minimum interface required to work as a generic part of the computation graph. RDataFrames and results of transformations can be cast to this type via ROOT::RDF::ToCommonNodeType.
