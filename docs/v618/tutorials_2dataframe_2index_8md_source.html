3 \brief These examples show the functionalities of [RDataFrame](classROOT_1_1RDataFrame.html): ROOT's declarative analysis interface.
4 
5 ROOT's [RDataFrame](classROOT_1_1RDataFrame.html) offers a high level interface for analyses of data stored in [TTree](classTTree.html)s, [CSV files](classROOT_1_1RDF_1_1RCsvDS.html) and [other data formats](classROOT_1_1RDF_1_1RDataSource.html).
6 
7 In addition, multi-threading and other low-level optimisations allow users to exploit all the resources available on their machines transparently.
8 
9 In a nutshell:
10 ~~~{.cpp}
11 ROOT::EnableImplicitMT(); // Tell ROOT you want to go parallel
12 ROOT::RDataFrame d("myTree", "file_*.root"); // Interface to TTree and TChain
13 auto myHisto = d.Histo1D("Branch_A"); // This happens in parallel!
14 myHisto->Draw();
15 ~~~
16 
17 Explore the examples below or go to [RDataFrame user guide](classROOT_1_1RDataFrame.html).
