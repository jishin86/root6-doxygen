◆ Open()

TArchiveFile * TArchiveFile::Open ( const char *  url,
TFile *  file 
)
static

Return proper archive file handler depending on passed url.

The handler is loaded via the plugin manager and is triggered by the extension of the archive file. In case no handler is found 0 is returned. The file argument is used to access the archive. The archive should be specified as url with the member name as the anchor, e.g. "root://pcsalo.cern.ch/alice/event_1.zip#tpc.root", where tpc.root is the file in the archive to be opened. Alternatively the sub-file can be specified via its index number, e.g. "root://pcsalo.cern.ch/alice/event_1.zip#3". This function is normally only called via TFile::Open().
