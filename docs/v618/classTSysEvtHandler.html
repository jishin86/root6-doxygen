◆ DeActivate()

void TSysEvtHandler::DeActivate ( )

De-activate a system event handler.

Use this method to temporarily disable an event handler to avoid it from being recursively called. Use DeActivate() / Activate() instead of Remove() / Add() for this purpose, since the Add() will add the handler back to the end of the list of handlers and cause it to be called again for the same, already handled, event.
