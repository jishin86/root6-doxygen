TEvePointSet Class Reference
Graphics » 3D Graphics » Event Display

TEvePointSet is a render-element holding a collection of 3D points with optional per-point TRef and an arbitrary number of integer ids (to be used for signal, volume-id, track-id, etc).

3D point representation is implemented in base-class TPolyMarker3D. Per-point TRef is implemented in base-class TPointSet3D.

By using the TEvePointSelector the points and integer ids can be filled directly from a TTree holding the source data. Setting of per-point TRef's is not supported.

TEvePointSet is a TEveProjectable: it can be projected by using the TEveProjectionManager class.
