template<class ScalarType = double>
class ROOT::Math::PxPyPzE4D< ScalarType >

Class describing a 4D cartesian coordinate system (x, y, z, t coordinates) or momentum-energy vectors stored as (Px, Py, Pz, E).

The metric used is (-,-,-,+)
