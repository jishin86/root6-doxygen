TBits Class Reference
Core ROOT classes » Containers

Container of bits.

This class provides a simple container of bits. Each bit can be set and tested via the functions SetBitNumber and TestBitNumber.The default value of all bits is kFALSE. The size of the container is automatically extended when a bit number is either set or tested. To reduce the memory size of the container use the Compact function, this will discard the memory occupied by the upper bits that are 0.
