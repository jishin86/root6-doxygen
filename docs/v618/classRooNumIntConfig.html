RooNumIntConfig Class Reference
Roofit » Roofitcore

RooNumIntConfig holds the configuration parameters of the various numeric integrators used by RooRealIntegral.

RooRealIntegral and RooAbsPdf use this class in the (normalization) integral configuration interface
