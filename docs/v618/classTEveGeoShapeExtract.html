TEveGeoShapeExtract Class Reference
Graphics » 3D Graphics » Event Display

Globally positioned TGeoShape with rendering attributes and an optional list of daughter shape-extracts.

Vessel to carry hand-picked geometry from gled to reve. This class exists in both frameworks.
