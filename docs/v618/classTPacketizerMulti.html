TPacketizerMulti Class Reference
PROOF » PROOF kernel Libraries

This class allows to do multiple runs in the same query; each run can be a, for example, different dataset or the same dataset with entry list.

The multiple packetizer contains a list of packetizers which are processed in turn. The bit TSelector::kNewRun is set in the TSelector object when a new packetizer is used.
