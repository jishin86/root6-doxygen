4 The **Minuit2** library is a new object-oriented implementation, written in C++,
5 of the popular MINUIT minimization package. These new version provides basically
6 all the functionality present in the old Fortran version, with almost equivalent
7 numerical accuracy and computational performances. Furthermore, it contains new
8 functionality, like the possibility to set single side parameter limits or the
9 FUMILI algorithm, which is an optimized method for least square and log likelihood
10 minimizations. The package has been originally developed by M. Winkler and F. James.
11 More information on the new C++ version can be found on the
12 [MINUIT Web Site](http://www.cern.ch/minuit).
13 
14 Minuit2, originally developed in the SEAL project, is now distributed within %ROOT.
15 The API has been then changed in this new version to follow the %ROOT coding convention
16 (function names starting with capital letters) and the classes have been moved inside
17 the namespace _ROOT::Minuit2_. In addition, the %ROOT distribution contains classes
18 needed to integrate Minuit2 in the %ROOT framework.
19 
20 A new class has been introduced, ROOT::Minuit2::Minuit2Minimizer, which implements
21 the interface ROOT::Math::Minimizer. Within %ROOT, it can be instantiates also using
22 the %ROOT plug-in manager. This class provides a convenient entry point for using Minuit2\.
23 An example of using this interface is the %ROOT tutorial _tutorials/fit/NumericalMinimization.C_
24 or the Minuit2 test program
25 [<tt>testMinimize.cxx</tt>](https://github.com/cxx-hep/root-cern/blob/master/math/minuit2/test/testMinimize.cxx).
26 
27 A standalone version of Minuit2 (independent of %ROOT) can be downloaded from
28 [here](http://www.cern.ch/mathlibs/sw/5_34_14/Minuit2/Minuit2-5.34.14.tar.gz).
29 It does not contain the %ROOT interface and it is therefore
30 totally independent of external packages and can be simply build using the _configure_ script
31 and then _make_. Example tests are provided in the directory _test/MnSim_ and _test/MnTutorial_
32 and they can be built with the _make check_ command.
33 The [Minuit2 User Guide](https://root.cern.ch/root/htmldoc/guides/minuit2/Minuit2.html)
34 provides all the information needed for using directly (without add-on packages like %ROOT) Minuit2.
