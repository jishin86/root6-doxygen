4 /// Palette coloring for histogram is activated thanks to the options `PFC`
5 /// (Palette Fill Color), `PLC` (Palette Line Color) and `AMC` (Palette Marker Color).
6 /// When one of these options is given to `TH1::Draw` the histogram get its color
7 /// from the current color palette defined by `gStyle->SetPalette(...)`. The color
8 /// is determined according to the number of objects having palette coloring in
9 /// the current pad.
10 ///
11 /// In this example five histograms are displayed with palette coloring for lines and
12 /// and marker. The histograms are drawn with makers and error bars and one can see
13 /// the color of each histogram is picked inside the default palette `kBird`.
