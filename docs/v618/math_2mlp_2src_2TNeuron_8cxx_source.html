13 //
14 // TNeuron
15 //
16 // This class decribes an elementary neuron, which is the basic
17 // element for a Neural Network.
18 // A network is built connecting neurons by synapses.
19 // There are different types of neurons: linear (a+bx),
20 // sigmoid (1/(1+exp(-x)), tanh or gaussian.
21 // An external function can also be used, together with its derivative.
22 // In a Multi Layer Perceptron, the input layer is made of
23 // inactive neurons (returning the normalized input) and output neurons
24 // are linear. Hidden neurons may be anything, the default being sigmoids.
25 //
26 // This implementation contains several methods to compute the value,
27 // the derivative, the DeDw, ...
28 // Values are stored in local buffers. The SetNewEvent() method is
29 // there to inform buffered values are outdated.
30 //
